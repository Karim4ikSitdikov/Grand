# Быстрый старт

## Шаг 1: Настройка конфигурации

Откройте файл `js/config.js` и обновите контактную информацию:

```javascript
contacts: {
    phone: '+7 (495) 123-45-67',        // Ваш телефон
    phoneRaw: '74951234567',            // Телефон без форматирования
    email: 'info@urgrand.ru',           // Ваш email
    address: 'г. Москва, ул. Примерная, д. 1', // Ваш адрес
    mainSite: 'https://urgrand.ru'      // Ссылка на основной сайт
}
```

## Шаг 2: Добавление изображений

1. **Логотип компании**: 
   - Скачайте логотип с https://urgrand.ru/
   - Сохраните как `images/logo.png`

2. **Логотипы банков**:
   - Скачайте SVG логотипы банков
   - Сохраните в `images/banks/` с именами:
     - `sberbank.svg`
     - `vtb.svg`
     - `alfa.svg`
     - `tinkoff.svg`
     - `raiffeisen.svg`
     - `gazprombank.svg`

## Шаг 3: Настройка WhatsApp

В файле `js/config.js` номер уже настроен из контактов. Убедитесь, что `phoneRaw` указан правильно.

## Шаг 4: Настройка Telegram (опционально)

1. Создайте бота через [@BotFather](https://t.me/BotFather)
2. Получите токен
3. Получите Chat ID (можно через [@userinfobot](https://t.me/userinfobot))
4. Обновите в `js/config.js`:
   ```javascript
   telegram: {
       botToken: 'ВАШ_ТОКЕН',
       chatId: 'ВАШ_CHAT_ID',
       enabled: true
   }
   ```

## Шаг 5: Настройка Bitrix (если используется)

Следуйте инструкциям в файле `BITRIX_SETUP.md`

## Шаг 6: Тестирование

1. Откройте `index.html` в браузере
2. Проверьте все формы
3. Проверьте контакты
4. Протестируйте отправку заявок

## Готово!

Сайт готов к использованию. Загрузите файлы на веб-сервер или используйте локально.

## Дополнительные настройки

### Изменение цветов

Цвета можно изменить в файле `css/styles.css` в секции `:root`:

```css
:root {
    --primary-color: #1E3A8A;  /* Основной синий цвет */
    --primary-dark: #1E40AF;    /* Темный синий */
    --primary-light: #3B82F6;   /* Светлый синий */
    /* ... */
}
```

### Изменение текстов

Все тексты находятся в файле `index.html`. Найдите нужный раздел и обновите текст.

### Добавление отзывов

Отзывы находятся в секции "Отзывы клиентов" в `index.html`. Добавьте новые блоки `.review-card` по образцу существующих.

### Добавление вопросов в FAQ

Вопросы находятся в секции FAQ в `index.html`. Добавьте новые блоки `.faq-item` по образцу существующих.
