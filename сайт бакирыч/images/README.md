# Изображения

## Необходимые файлы

### Логотип компании
- **logo.png** - Логотип Юридической Компании «Гранд»
  - Рекомендуемый размер: 200x50px (или пропорционально)
  - Формат: PNG с прозрачным фоном
  - Можно взять с основного сайта https://urgrand.ru/

### Логотипы банков

Все логотипы банков должны быть в формате SVG для лучшего качества:

- **banks/sberbank.svg** - Сбербанк
- **banks/vtb.svg** - ВТБ
- **banks/alfa.svg** - Альфа-Банк
- **banks/tinkoff.svg** - Тинькофф Банк
- **banks/raiffeisen.svg** - Райффайзенбанк
- **banks/gazprombank.svg** - Газпромбанк

## Где найти логотипы

1. Официальные сайты банков (раздел для партнеров/прессы)
2. Сайты конкурентов (как пример оформления)
3. Открытые источники с логотипами компаний

## Альтернатива

Если SVG логотипы недоступны, можно использовать PNG:
- Размер: примерно 150x40px
- Формат: PNG с прозрачным фоном
- Обновите расширения в `index.html` с `.svg` на `.png`
