# Инструкция по копированию изображений с основного сайта

## Логотип компании

1. Откройте https://urgrand.ru/
2. Откройте инструменты разработчика (F12)
3. Перейдите на вкладку Network (Сеть)
4. Обновите страницу (F5)
5. Найдите файл логотипа (обычно logo.png или logo.svg)
6. Сохраните его как `images/logo.png`

Или используйте прямую ссылку:
- Попробуйте: https://urgrand.ru/local/templates/main/img/logo.png
- Или найдите в исходном коде страницы тег `<img>` с логотипом

## Баннеры и другие изображения

1. На основном сайте найдите нужные изображения
2. Правой кнопкой мыши → "Сохранить изображение как..."
3. Сохраните в папку `images/urgrand/`

## Логотипы банков

Логотипы банков можно найти:
1. На официальных сайтах банков
2. В разделе для партнеров/прессы
3. Используйте SVG формат для лучшего качества

Рекомендуемые источники:
- Сбербанк: https://www.sberbank.ru/ru/press
- ВТБ: https://www.vtb.ru/o-banke/press-tsentr/
- Альфа-Банк: https://alfabank.ru/about/press/
- Тинькофф: https://www.tinkoff.ru/about/press/
- Райффайзен: https://www.raiffeisen.ru/about/press/
- Газпромбанк: https://www.gazprombank.ru/about/press/

## Автоматическое копирование (опционально)

Можно использовать инструменты для скачивания:
- wget
- curl
- Browser extensions (например, Image Downloader)

Пример команды:
```bash
# Для логотипа
curl -o images/logo.png "https://urgrand.ru/path/to/logo.png"
```
