# Структура проекта

## Описание файлов

### Основные файлы

- **index.html** - Главная страница лендинга со всеми секциями
- **README.md** - Основная документация проекта
- **QUICKSTART.md** - Инструкция по быстрому старту
- **BITRIX_SETUP.md** - Инструкция по настройке Bitrix
- **.gitignore** - Исключения для Git

### CSS

- **css/styles.css** - Все стили сайта, включая:
  - Адаптивный дизайн
  - Анимации
  - Модальные окна
  - Формы

### JavaScript

- **js/config.js** - Конфигурация сайта (контакты, настройки интеграций)
- **js/script.js** - Основная логика:
  - Обработка форм
  - FAQ аккордеон
  - Модальные окна
  - Интеграции (Bitrix, WhatsApp, Telegram)
  - Валидация и маски ввода

### PHP (для Bitrix)

- **local/ajax/form_handler.php** - Обработчик форм для Bitrix CRM

### Изображения

- **images/** - Папка для изображений

## Секции лендинга

1. **Header** - Шапка с логотипом, контактами и кнопкой заказа звонка
2. **Hero** - Главный баннер с заголовком и преимуществами
3. **Banks** - Логотипы банков
4. **Main Form** - Основная форма-анкета для оценки случая
5. **Services** - Блок с описанием услуг
6. **Reviews** - Отзывы клиентов и судебная практика
7. **FAQ** - Часто задаваемые вопросы
8. **Contact Form** - Форма обратной связи с кнопками мессенджеров
9. **Contacts** - Контактная информация
10. **Footer** - Подвал сайта

## Интеграции

### Bitrix CRM
- Автоматическое создание лидов
- Сохранение всех данных анкеты
- Настройка через `local/ajax/form_handler.php`

### WhatsApp
- Отправка заявок через WhatsApp Web
- Настройка номера в `js/config.js`

### Telegram
- Отправка уведомлений через Telegram Bot API
- Настройка токена и chat ID в `js/config.js`
